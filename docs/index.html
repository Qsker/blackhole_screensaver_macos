📥 安装

  1. 从 Releases 下载 BlackHoleSaver.saver.zip 并解压
  2. 双击 BlackHoleSaver.saver 安装
  3. 打开系统设置 → 屏幕保护程序，选择 BlackHoleSaver
  4. 授予屏幕录制权限（见下方）

🔒 macOS 27 — 录屏权限

系统设置 → 隐私与安全性 → 屏幕与系统音频录制 → 点 + → 前往 /System/Library/CoreServices/ → 选 WallpaperAgent.app → 打开开关

🔒 macOS 14 / 15 — 录屏权限

系统设置 → 隐私与安全性 → 屏幕与系统音频录制 → 从 Finder 把 /System/Library/Frameworks/ScreenSaver.framework/PlugIns/legacyScreenSaver.appex 拖入列表 → 打开开关

没有权限也能用，黑洞会吞星空而不是桌面。
