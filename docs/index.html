📥 安装

  1. 从 Releases 下载并解压
  2. 双击 BlackHoleSaver.saver 安装
  3. 系统设置 → 屏幕保护程序 → BlackHoleSaver
  4. 授予屏幕录制权限

🔒 macOS 27

隐私 → 屏幕录制 → + → /System/Library/CoreServices/WallpaperAgent.app

🔒 macOS 14 / 15

拖入 ScreenSaver.framework/PlugIns/legacyScreenSaver.appex
